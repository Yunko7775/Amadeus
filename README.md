# Amadeus  System

A sophisticated AI interaction system inspired by **Amadeus** from the *Steins;Gate* series. This project integrates a Live2D interactive interface, Large Language Models (LLM), and dynamic Text-to-Speech (TTS) to create an AI assistant with long-term memory and emotional intelligence.

##  Key Features

* **Live2D Interaction**: Features a high-fidelity Makise Kurisu model with real-time motion tracking and facial expression transitions.
* **Long-term Memory Management**: Implements persistent user-data storage via `memory_manager.py`, allowing the AI to remember user preferences and past conversations.
* **Neural TTS**: Powered by GPT-SoVITS technology for high-accuracy character voice synthesis.
* **Lightweight Backend**: Built on Flask to handle seamless communication between the frontend UI and AI models.

## 🛠️ Tech Stack

* **Core Language**: Python 
* **GUI Framework**: Live2D SDK (Tkinter)
* **Backend**: Flask
* **AI Logic**: Gemini API (Configurable)
* **Audio Synthesis**: GPT-SoVITS-v2pro
